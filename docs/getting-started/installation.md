# Instalación

Para utilizar `apklis-downloader`, necesitas tener Python instalado en tu ordenador. Se recomienda utilizar Python 3.7 o superior.

## Instalación desde PyPI

Puedes instalar la última versión estable directamente desde PyPI usando `pip`:

```bash
pip install apklis-downloader
```

## Instalación desde Código Fuente (GitHub)

Si quieres instalar la versión más reciente en desarrollo:

```bash
git clone https://github.com/ElJoker63/apklis-downloader.git
cd apklis-downloader
pip install -e .
```

## Dependencias

La librería utiliza `httpx` para realizar peticiones HTTP asíncronas de alto rendimiento y streaming eficiente de archivos grandes:

- `httpx>=0.20.0`
